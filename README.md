# Face Recognition System

A Python-based Face Recognition System using OpenCV and LBPH Face Recognizer.

## Features
- Face Detection
- Face Recognition
- GUI using Tkinter
- Camera Support

## Technologies
- Python
- OpenCV
- Pillow
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python face_recognition_gui.py
```
